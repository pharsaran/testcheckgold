### TC-001: การเข้าใช้งานพร้อมกัน (Concurrent Access)

**รายละเอียด:**
- ระบบรองรับผู้ใช้หลายคนพร้อมกัน
- Socket.io รองรับหลาย WebSocket connections
- Express API รองรับ concurrent HTTP requests

**การทดสอบ:**
```bash
วิธีที่ 1: ทดสอบ API (HTTP Requests)
node test/test-api.js

วิธีที่ 2: ทดสอบ WebSocket (แนะนำ)
node test/test-concurrent-websocket.js

```

**ผลการทดสอบ:**
- จำลองผู้ใช้ 10 คนเข้าใช้งานพร้อมกัน
- แสดงผลผู้ใช้แต่ละคน (User-01 ถึง User-10)
- ตรวจสอบสถานะการเชื่อมต่อและข้อมูลที่ได้รับ

### TC-002: ความเสถียรของธุรกรรม (Transaction Stability)

**รายละเอียด:**
- ทดสอบ Transaction ของ Gold 99.99%  100 รายการพร้อมกัน
- Transaction service รองรับการสร้างพร้อมกัน
- **Test Result:** สร้าง 100 transactions 

**การทดสอบ:**
```bash
node test/test-api.js
# Result: Created 100 transactions successfully
# Retrieved 100 transactions
```

**ข้อมูล Transaction:**
- Symbol: GOLD9999 (ระบุว่าเป็น Gold 99.99%)
- Price: ใช้ราคาจริงจากระบบ (ราคาขายสำหรับ Buy, ราคาซื้อสำหรับ Sell)
- State: Buy/Sell
- Date/Time: ISO timestamp

**ผลการทดสอบ:**
- จำลองการทำรายการซื้อขาย 100 รายการพร้อมกัน
- แสดงรายละเอียดแต่ละรายการ (10 รายการแรกและ 10 รายการสุดท้าย)

### TC-003: การควบคุมสถานะราคา (Price Status Control)
**สถานะ:** ผ่าน ✅

**รายละเอียด:**
- สามารถทดสอบปุ่ม Status (Online, Pause, Stop) ของราคาทองคำได้
- รองรับ 3 ประเภทราคา: Gold Spot, Gold 99.99%, Gold 96.50%

**การทดสอบ:**
```bash
node test/test-api.js
```

**ผลการทดสอบ:**
- สถานะ Online: ระบบอัปเดตราคาอัตโนมัติทุก 10 วินาที ✅
- สถานะ Pause: ระบบหยุดอัปเดตราคา แต่คงราคาเดิม ✅
- สถานะ Stop: ระบบตั้งราคาเป็น 0 ทันที ✅
- สถานะผสม: แต่ละประเภทราคามีสถานะต่างกันได้ ✅

**UI Testing:**
- เปิด `http://localhost:3000`
- กดปุ่ม "เปิดการควบคุมสถานะ"
- เลือกสถานะ Online/Pause/Stop
- กดปุ่ม Save

### TC-004: การแสดงผล RealTime

**รายละเอียด:**
- ระบบสามารถแสดงผลราคาทองคำแบบ RealTime ได้
- WebSocket (Socket.io) พร้อมใช้งาน
- Broadcast priceUpdate ทุก 10 วินาที
- Frontend รับและอัปเดต UI อัตโนมัติ

**การทดสอบ:**
```bash
node test/test-api.js

# วิธีที่ 2: ทดสอบผ่าน Browser
# 1. รันเซิร์ฟเวอร์: npm start
# 2. เปิด browser: http://localhost:3000
# 3. ตรวจสอบ:
#    - ราคาอัปเดตอัตโนมัติทุก 10 วินาที
#    - WebSocket connection สำเร็จ
#    - ราคาแสดงผลแบบ real-time
```

**ผลการทดสอบ:**
- WebSocket Connection: สำเร็จ ✅
- Initial Data Event: รับได้ ✅
- Price Update Event: รับได้ (RealTime ทำงาน) ✅
- Status Update Event: รับได้ ✅
- New Transaction Event: รับได้ (RealTime ทำงาน) ✅

**Technical Details:**
- WebSocket: Socket.io
- Update Interval: 10 seconds
- Events: `initialData`, `priceUpdate`, `statusUpdate`, `newTransaction`
